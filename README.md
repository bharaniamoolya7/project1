# Data Analytics Project 1 – Data Cleaning & Preparation

## 📌 Overview

This project focuses on cleaning and preparing a raw dataset to improve data quality and ensure it is ready for analysis. The dataset was examined for missing values, duplicate records, and inconsistent data formats. After cleaning, the dataset became consistent, accurate, and suitable for further data analysis.

---

## 🎯 Project Objective

The objectives of this project are to:

- Identify missing or null values
- Remove duplicate records
- Standardize data formats
- Improve data consistency and accuracy
- Prepare the dataset for analysis

---

## 🛠️ Tools Used

- Microsoft Excel
- GitHub

---

## 📂 Dataset Information

**Dataset Name:** Dataset for Data Analytics

The dataset contains **1,200 records** and **14 columns** representing customer orders in an e-commerce system.

### Dataset Features

- **OrderID** – Unique order identifier
- **Date** – Order date
- **CustomerID** – Unique customer identifier
- **Product** – Product purchased
- **Quantity** – Number of items ordered
- **UnitPrice** – Price per item
- **ShippingAddress** – Delivery address
- **PaymentMethod** – Payment method used
- **OrderStatus** – Current order status
- **TrackingNumber** – Shipment tracking number
- **ItemsInCart** – Number of items in the cart
- **CouponCode** – Applied discount coupon
- **ReferralSource** – Customer referral source
- **TotalPrice** – Total order amount

### Dataset Size

- **Rows:** 1,200
- **Columns:** 14

---

## 🔧 Data Cleaning Process

### 1. Missing Values
- Identified missing values in the dataset.
- Filled or handled missing entries appropriately.

### 2. Duplicate Records
- Checked for duplicate records.
- Removed duplicate entries to maintain data integrity.

### 3. Data Formatting
- Standardized date formats.
- Corrected inconsistent text values.
- Verified numeric data types.

### 4. Validation
- Confirmed that duplicate records were removed.
- Verified consistent formatting across the dataset.

---

## 📊 Results

After completing the data cleaning process:

- Missing values were handled successfully.
- Duplicate records were removed.
- Data formats were standardized.
- The dataset became clean, consistent, and ready for analysis.
